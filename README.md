一、下载此代码

二、安装
pip install -r requirements.txt
pip install pywin32

三、安装同花顺软件
https://mams.10jqka.com.cn/new/server/html/61557.html

四、运行
1. 修改账号信息： client_config.json
2. 修改同花顺安装路径：easytrader/config/client.py
3. python main.py

注意事项
1. 需要先注册同花顺账号;
2. 需要先打开xiadan.exe，并登陆;
3. 需要保持xiadan.exe程序为运行状态。


### 支持券商

* 海通客户端(海通网上交易系统独立委托)
* 华泰客户端(网上交易系统（专业版Ⅱ）)
* 国金客户端(全能行证券交易终端PC版)
* 其他券商通用同花顺客户端(需要手动登陆)
